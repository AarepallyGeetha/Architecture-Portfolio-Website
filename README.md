# Advanced CSS3 & Responsive Architecture Portfolio Website

## Overview
A modern portfolio website built using Semantic HTML5, CSS Grid, Flexbox, CSS Variables, and Responsive Design principles.

## Features
- Semantic HTML5 Structure
- CSS Grid Layout
- Flexbox Navigation
- Mobile-First Design
- Responsive Media Queries
- Light/Dark Theme Toggle
- Clean User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Outcome
A pixel-perfect responsive portfolio website that adapts seamlessly across mobile, tablet, and desktop devices while supporting dynamic theme switching.
